# MarTech ERP Frontend

Next.js frontend for MarTech ERP system.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` (optional):
```env
NEXT_PUBLIC_API_URL=http://localhost:8000
```

3. Run development server:
```bash
npm run dev
```

The app will be available at http://localhost:3000

## Login

Use any email/password combination. The backend accepts any credentials for mock authentication.

Default test user: `admin@marteche.com` (any password)
